1. Basic if statement:
if [ condition ]
then
  # commands
fi
2. If-else statement:
if [ condition ]
then
  # commands
else
  # commands
fi
3. If-elif-else statement:
if [ condition ]
then
  # commands
elif [ condition ]
then
  # commands
else
  # commands
fi
4. Nested if statement:
if [ condition ]
then
  if [ condition ]
  then
    # commands
  fi
fi
